Git Initialization & Push To GitHub

Follow these step-by-step terminal commands to initialize a new Git repository, add files, commit them, and push to a main branch on GitHub.

1. Initialize a new Git repository

Creates a new, empty Git repository in your current directory.

git init

2. Stage all files

Adds all files in the current directory to the staging area, preparing them for the next commit.

git add .

3. Commit the changes

Commits the staged files with the specified message.

git commit -m "Initial Build"

4. Rename the default branch

Ensures the primary branch is named main, which is the current standard for GitHub.

git branch -M main

5. Add remote repository URL

Links your local repository to a remote repository on GitHub. (Replace the URL with your actual GitHub repository URL.)

git remote add origin https://github.com/your-username/your-repository.git

6. Push code to GitHub

Pushes the commits to the remote main branch and sets the upstream tracking branch.

git push -u origin main

Are these all the files to be created?

No, these are not files. These are commands you run in your terminal (like Command Prompt, PowerShell, or Git Bash). They apply to the files you already have in your project folder (like your HTML, CSS, or JS files).

How to Automate or "Run at Once"

Yes, you can automate this! You have two main options:

Option 1: Chain them in the terminal (Quickest)

You can use the && operator to run all the commands back-to-back in one go. If one fails, the rest will stop.

git init && git add . && git commit -m "Initial Build" && git branch -M main && git remote add origin https://github.com/your-username/your-repository.git && git push -u origin main

Option 2: Create a Script File (Best for reuse)

If you do this often, you can create a script file in your project folder that runs these commands for you.

  • On Windows: Create a new file named push.bat
  • On Mac/Linux: Create a new file named push.sh

Paste the following code into the file (don't forget to replace the URL!). Whenever you double-click the file or run it in the terminal, it will do everything instantly.

git init
git add .
git commit -m "Initial Build"
git branch -M main
git remote add origin https://github.com/your-username/your-repository.git
git push -u origin main
pause
